## üöÄ Local Development

Want to run this project locally? It's simple. Since it's a static site, you just need a local server to handle the assets correctly.

1. Clone the repository:
   git clone https://github.com/vinayakgawade0/vg-resume.git
   cd vg-resume

2. Start a local server:
   python -m http.server 8000

3. View the site:
   Open your browser and go to: http://localhost:8000/index.html
